# Heart - CSS Hover Effect

Este foi um dos meus primeiros projetos usando **HTML** e **CSS**.

A ideia é simples, mas especial:
um quadrado que, ao passar o mouse por cima, se transforma em um coração.

Criei esse projeto como uma surpresa para minha namorada, então ele tem um significado muito importante pra mim — representa o início da minha jornada na programação.

---

## Demonstração

Passe o mouse sobre o quadrado e veja ele se transformar em um coração.

---

## Tecnologias utilizadas

* HTML5
* CSS3

---

## O que pratiquei

* Uso de `:hover`
* Transformações com `transform`
* Criação de formas com CSS
* Uso de pseudo-elementos (`::before` e `::after`)
